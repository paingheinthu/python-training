# Exercises: File Read/Write & File Operations

## Basic File Reading Exercises

1. Create a text file called `greeting.txt` with the content "Hello, Python!". Read and print it.

2. Create a file with your name and read it back.

3. Read a file line by line and print each line with its line number.

4. Read a file and count the total number of lines.

5. Read a file and count the total number of characters (including spaces).

6. Read a file and count the number of words.

7. Create a file with multiple lines, then read only the first 3 lines.

8. Read a file and print only lines that contain the letter 'a'.

9. Read a file and convert all text to uppercase.

10. Read a file and remove empty lines when printing.

---

## File Writing Exercises

11. Write your name, age, and city to a file using separate `write()` calls.

12. Write the numbers 1 to 10 to a file, each on a new line.

13. Write the multiplication table of 7 to a file.

14. Write a list of fruits to a file, one per line.

15. Create a file with the following format:
    ```
    Name: Alice
    Age: 30
    City: New York
    ```

16. Write all even numbers from 1 to 20 to a file.

17. Write a pattern to a file:
    ```
    *
    **
    ***
    ****
    *****
    ```

18. Write the squares of numbers 1 to 10 to a file.

19. Write a CSV header row to a file: `Name,Age,Salary`

20. Create a file with a list of colors, each on a new line.

## Project-based Exercises

### Project 1: Student Grade Manager
- Write student names and grades to a file
- Read the file and calculate the average grade
- Append new student records
- Find the student with the highest grade

### Project 2: Personal Diary
- Create a program that appends diary entries to a file with dates
- Read and display all entries
- Search for entries containing specific keywords

### Project 3: Inventory System
- Create a CSV file with product information (name, quantity, price)
- Read the inventory
- Update quantities and save back
- Generate a report file with total inventory value

### Project 4: Word Frequency Analyzer
- Read a text file
- Count word frequencies
- Write the results to a new file sorted by frequency

### Project 5: Log File Analyzer
- Create a log file with entries
- Read and filter entries by date or error level
- Generate a summary report
- Save the report to a file
