1. Computer History

Early Devices

  • Abacus (3000 BC) – First known calculating tool
  • Mechanical Calculators (1600s) – Designed by Pascal & Leibniz

First Computers

  • Charles Babbage (1800s) – Designed the Analytical Engine (first computer design)
  • Ada Lovelace – First computer programmer

Electronic Computers

  • ENIAC (1945) – First large-scale electronic computer
  • UNIVAC (1951) – First commercial computer

Modern Computing

  • 1976 – Apple I
  • 1981 – IBM PC
  • 2007 – Smartphones revolution

3rd Generation – High-Level Languages

  • C (1972)

4th Generation – Declarative Languages

SQL, MATLAB

5th Generation – AI/Logic-Based

Prolog, rule-based systems

Functional Programming

Lisp, Haskell, Elixir

Declarative Programming

SQL, HTML, CSS

Compiled vs. Interpreted Languages

Programming Data Types

Data types define the kind of value a variable can hold. The size and range can vary by language and architecture.

Type Common Size Range / Precision
Boolean 1 byte true / false
char 1 byte -128 to 127 or 0 to 255
int8 / byte 1 byte -128 to 127
uint8 / ubyte 1 byte 0 to 255
int16 / short 2 bytes -32,768 to 32,767
uint16 / ushort 2 bytes 0 to 65,535
int32 / int 4 bytes -2,147,483,648 to 2,147,483,647
uint32 / uint 4 bytes 0 to 4,294,967,295
int64 / long 8 bytes -9,223,372,036,854,775,808 to 9,223,372,036,854,775,807
uint64 / ulong 8 bytes 0 to 18,446,744,073,709,551,615
float 4 bytes ~6-9 decimal digits of precision
double 8 bytes ~15-17 decimal digits of precision

Python Data Types & Ranges

Python int: unlimited precision
Float: IEEE 754 double precision (64-bit)

Variable Naming Rules

Meaningful, no spaces, cannot start with number, cannot use keywords.

Casing Styles

CamelCase: variableName
Snake case: variable_name
Pascal case: ClassName
CONST: CONSTANT_NAME
Kebab: file-name

Casing by Language

Python: variable_name, ClassName, CONSTANT_NAME, function_name, file_name

PHP: variableName, functionName, ClassName, FileName, CONSTANT_NAME, json_key, db_key

Go: localVariable, PublicVariable, functionName, PublicFunction, file_name, CONSTANT_NAME
